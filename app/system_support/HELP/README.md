# HELP SECTION

HELP is the internal help center of the app.

It stores explanations, examples, troubleshooting notes, and command guidance so the assistant can explain the system without guessing.

## Main idea

The app should be able to answer things like:

- EXPLAIN MEMORY
- EXPLAIN LLM
- EXPLAIN RUNTIME
- HELP COMMANDS
- HELP TROUBLESHOOTING
- HOW DO I USE NOTES?
